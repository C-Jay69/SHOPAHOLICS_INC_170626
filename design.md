# Shopaholics Inc — Design System

## Brand Voice
Professional, customer-obsessed, with playful sarcasm. Witty UI copy. Never corporate, never cringe.

## Typography
- **Font Family:** Poppins (Google Fonts) — display + body
- **Headings:** Bold (700–900), gradient text (purple→cyan)
- **Body:** 400–500 weight, gray-300 on dark backgrounds
- **Sizes:** text-5xl lg:text-7xl for hero, text-2xl for section heads, text-sm for labels

## Color Palette
- **Background Primary:** #000000 (black)
- **Background Secondary:** #0a0a0a
- **Background Card:** #111111 / #1a1a1a
- **Background Elevated:** #2a2a2a
- **Border:** #1f2937 (gray-800), subtle
- **Text Primary:** #ffffff
- **Text Secondary:** #d1d5db (gray-300)
- **Text Muted:** #6b7280 (gray-500)
- **Accent Purple:** #a855f7 (purple-500)
- **Accent Cyan:** #06b6d4 (cyan-500)
- **Gradient:** from-purple-500 to-cyan-500 (buttons, headings, badges)

## Buttons
- **Primary:** bg-gradient-to-r from-purple-500 to-cyan-500, white text, rounded-lg
- **Secondary:** border-2 border-white/20, white text, hover: bg-white text-black

## Cards
- bg: #111111
- border: 1px solid #1f2937
- border-radius: rounded-xl
- hover: border-purple-500/50

## No-Nos
- No animations per spec
- No neon chaos — use gradients tastefully
- No popups for marketing
